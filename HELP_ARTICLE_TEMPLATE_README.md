## 使用方法

### 一键生成完整文章
```bash
node help-article-complete-generator.js
```

### 单独生成文章模板
```bash
node help-article-template.js
```

### 批量生成测试（验证一致性）
```bash
node test-batch-generation.js
```

## 文件说明

### 核心文件
- `help-article-template.js` - 文章模板生成器
- `help-article-complete-generator.js` - 完整文章生成器（文章+封面）
- `title-only-cover-generator.js` - 标题专用封面生成器
- `test-batch-generation.js` - 批量测试脚本

### 输出文件
- `complete_help_article_[timestamp].json` - 完整文章数据
- `help_article_cover_[timestamp].png` - 封面图片
- `help_article_[timestamp].json` - 文章模板数据
